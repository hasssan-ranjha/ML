# 🚢 Titanic Survival Predictor

A machine learning application that predicts whether a Titanic passenger would have survived, based on details like class, age, sex, and fare. The project covers the full pipeline — from a trained Logistic Regression model to a clean, interactive desktop interface — built to explore how a model goes from training data to a working, usable tool.

## ✨ Features

- Predicts survival outcome from passenger details in real time
- Clean, modern desktop GUI — no command line needed to use it
- Fully offline, self-contained model loaded via Pickle

## ⚙️ Installation

```bash
git clone https://github.com/hasssan-ranjha/ML.git
cd ML/titanic-survival-predictor
pip install -r requirements.txt
```

## ▶️ Usage

```bash
python gui.py
```

Enter the passenger details in the GUI (class, sex, age, fare, etc.) and click **Predict** to see the result.
